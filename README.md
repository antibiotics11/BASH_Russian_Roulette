# BASH_Russian_Roulette

## Info (English)

Shell Script Russian Roulette for Linux. 

## Start Game

You can start the game by running "start.sh".

```
cd ./BASH_Russian_Roulette
sudo sh start.sh
```

### If Error Occured 

```
unlink /bin/sh
ln -s /bin/bash /bin/sh
```
